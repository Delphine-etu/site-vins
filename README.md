# site-vins
Projet universitaire consistant en un blog permettant à des utilisateurs de partager des notes et des commentaires concernant des vins. Le projet était à réaliser sous le framework express avec intégration du moteur de template nunjucks pour la présentation et de la base de données sqlite pour la persistance des données.
Ce projet a été réalisé individuellement et sans outil d'Intelligence Artificielle.
